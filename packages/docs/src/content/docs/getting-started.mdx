## Define a schema

```ts
import { sz, defineTable } from "surreal-zod";

export const User = defineTable("user", {
  email: sz.email().unique(),
  name: sz.string().optional(),
  createdAt: sz.date().default(() => new Date()),
});
```

## Generate + migrate

```sh
sz gen       # diff schema -> write up/down migration
sz migrate   # apply pending migrations
```

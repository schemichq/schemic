# schemic

The [Schemic](https://schemic.dev) CLI — schema-as-code migrations for any database.

This is a convenience alias for [`@schemic/cli`](https://www.npmjs.com/package/@schemic/cli): it lets
you run the CLI as `bunx schemic` / `npx schemic` (or install it for the `schemic` / `sc` bins). To
start a new project use `bun create schemic`; to add Schemic to an existing one, `schemic init`.
